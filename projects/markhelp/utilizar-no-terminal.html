Utilizando o MarkHelp no terminal

O MarkHelp foi escrito para funcionar em sistemas Linux. Por isso, o projeto acompanha um script para geração de pacotes Debian. Veja mais sobre isso clicando aqui.

Após a correta instalação, o terminal possíbilita uma utilização global do MarkHelp, sem a necessidade de programar. Isso é muito útil para quando a necessidade é apenas gerar uma documentação HTML e disponibilizá-la manualmente em algum canal na internet.

Obs: para automatizar o processo de criação de documentações, veja Utilizar como Biblioteca.

Dentro do terminal, basta digitar o comando markhelp --help para exibir a lista de ajuda.

markhelp --help
